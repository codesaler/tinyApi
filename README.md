# tinyApi

极其精简的 Api 脚手架。

### features

- 使用 composer
- DB：[catfan/Medoo](https://github.com/catfan/Medoo)
- Redis：[nrk/predis](https://github.com/nrk/predis)
- Log：[Seldaek/monolog](https://github.com/Seldaek/monolog)
- 不支持 RESTful
- 不支持自定义路由
- 默认 POST，参数强制使用 Json
- 默认开启 AES
- 默认开启 VERIFY_TIMESTAMP/VERIFY_HASH
